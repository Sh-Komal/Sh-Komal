**Next.js • React • REST APIs • Gemini AI**  
I architected this highly customized Next.js platform to fully supercharge my own job hunting process. It serves as an all-in-one productivity suite and demonstrates my ability to build complex internal tools.
- **Live Job Radar Feed:** Consumes rapid public APIs (Remotive, JSearch) to provide a centralized feed for global frontend roles with a custom "1-Click Apply" flow.
